# dbpmx

#### 介绍
动吧旅游-权限管理子系统


#### 软件架构
软件架构说明
技术栈：SpringBoot + Spring MVC + MyBatis + MySQL +Thymeleaf + jquery + shiro1. 

#### 安装教程

1.  安装jdk1.8
2.  安装maven
3.  安装mysql，导入dbsys.sql


#### 使用说明


1.  访问本地80端口，http：//localhost/doIndexUI
2.  可用的用户名admin，密码123


#### 参与贡献

1、使用SSM完成对部门、用户、菜单、角色的增删改查。
2、使用AspectJ实现aop异步记录日志，记录用户的操作。
3、使用shiro框架进行登录认证和权限控制。
